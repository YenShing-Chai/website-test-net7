## How to update

1. Add your image file to this folder (keep the same filename)
2. OR use the CMS (`net7-cms.html`) to update the path to any custom filename
3. Redeploy the site — upload both `net7.html` AND the `assets/` folder

## Tips

- PNG is best for images with transparent backgrounds (like the hero phones)
- JPG is best for photos (smaller file size)
- SVG is best for logos and icons (infinitely scalable)
- Keep images under 500KB for fast page loads
